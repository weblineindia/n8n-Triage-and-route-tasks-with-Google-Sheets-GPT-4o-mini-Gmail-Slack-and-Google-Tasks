# n8n-Triage-and-route-tasks-with-Google-Sheets-GPT-4o-mini-Gmail-Slack-and-Google-Tasks
This n8n workflow captures tasks via a webhook, validates and stores them in Google Sheets, automatically prioritizes pending tasks, creates urgent Google Tasks with Slack alerts, generates AI-powered summaries for remaining tasks using OpenAI, and emails a daily digest through Gmail.
